# zeing.github.io

Personal website for [Zeing](https://zeing.me) — built with Next.js 16 (Pages Router), React 19, and Tailwind CSS v4.

## Stack

- **Framework**: Next.js 16 (Pages Router)
- **Runtime**: Bun
- **Styling**: Tailwind CSS v4
- **Hosting**: Vercel

## Development

```bash
bun install
bun dev
```

## Deploy

Hosted on Vercel. Push to `main` triggers auto-deploy.
